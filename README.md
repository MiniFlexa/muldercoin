What is Muldercoin?
----------------

Muldercoin is a version of Litecoin with different block times, block reward and total coins.
 - 1 minute block targets
 - subsidy halves in 840k blocks (~1 years)
 - ~8.4 billion total coins
 - 5000 coins per block
